# Svelte Weather App

![Running example](./assets/running.gif)

## Prerequisites

Get API key from [OpenWeatherMap](https://openweathermap.org/).

## Environment

```sh
$ node -v
12.13.1
```

## To run

Modify key value in the `/src/key.js` file.

```
$ npm install
$ npm run dev
```

Access to `http://localhost:5000` using a browser.
